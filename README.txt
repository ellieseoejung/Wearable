1) Download the EXPO XDE if you have not already: https://expo.io/

2) Download and install react native swiper in the same directory as this: https://github.com/leecade/react-native-swiper

2) Open Expo XDE. It will prompt you to make an account

4) Click on "Project" on the top-left of the XDE, select "Open Project", and select this folder

5) Click on "Share" on the top-right of the XDE

6) Open the Expo app on your phone, scan the code, and the application will open

7) main.js is where all of the code for the app is. It is where we will be doing a lot of coding.
